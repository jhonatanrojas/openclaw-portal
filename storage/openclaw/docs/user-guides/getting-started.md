# 🚀 Primeros Pasos - OpenClaw Portal

## Bienvenido al OpenClaw Portal

El OpenClaw Portal es una interfaz web para gestionar y monitorear tu instancia de OpenClaw. Esta guía te ayudará a comenzar.

## 📋 Prerrequisitos

Antes de comenzar, asegúrate de tener:

1. **Acceso al portal**: `http://openclaw.deploymatrix.com/`
2. **Credenciales de acceso**:
   - Usuario: `admin@openclaw.test`
   - Contraseña: `password`
3. **Navegador compatible**: Chrome, Firefox, Safari, Edge (versiones recientes)

## 🔐 Primer Acceso

### Paso 1: Acceder al portal
1. Abre tu navegador web
2. Ve a: `http://openclaw.deploymatrix.com/`
3. Serás redirigido a la página de login

### Paso 2: Iniciar sesión
1. Ingresa tus credenciales:
   - **Email**: `admin@openclaw.test`
   - **Contraseña**: `password`
2. Haz clic en "Login"
3. Serás redirigido al dashboard principal

## 🛠️ Funcionalidades Básicas

### 1. **Documentación**
- **Buscar**: Encuentra documentación por palabra clave
- **Filtrar**: Filtra por categoría (instalación, configuración, etc.)
- **Crear**: Añade nueva documentación
- **Editar**: Modifica documentación existente

### 2. **Agentes**
- **Ver estado**: Monitorea el estado de los agentes
- **Asignar tareas**: Asigna tareas específicas a agentes
- **Ver logs**: Accede a los logs de actividad

### 3. **Tareas**
- **Kanban board**: Vista visual de tareas por estado
- **Filtros**: Filtra por prioridad, agente, estado
- **Seguimiento**: Monitorea el progreso en tiempo real

## 🔧 Configuración Inicial Recomendada

### 1. **Cambiar contraseña**
1. Ve a **Perfil** → **Configuración de cuenta**
2. Haz clic en "Cambiar contraseña"
3. Ingresa tu contraseña actual y la nueva
4. Guarda los cambios

### 2. **Configurar notificaciones**
1. Ve a **Configuración** → **Notificaciones**
2. Selecciona los tipos de notificaciones que deseas recibir
3. Configura los canales (email, in-app)
4. Guarda la configuración

### 3. **Personalizar dashboard**
1. En el dashboard, haz clic en "Personalizar"
2. Arrastra y suelta los widgets que necesites
3. Ajusta el tamaño y posición
4. Guarda el layout

## ❓ Solución de Problemas Comunes

### Problema: No puedo acceder al portal
**Solución:**
1. Verifica la URL: `http://openclaw.deploymatrix.com/`
2. Verifica tu conexión a internet
3. Intenta limpiar la cache del navegador
4. Contacta al administrador del sistema

### Problema: Error de login
**Solución:**
1. Verifica que estés usando las credenciales correctas
2. Asegúrate de que el usuario esté activo
3. Intenta restablecer la contraseña
4. Contacta al administrador si el problema persiste

### Problema: La página carga lentamente
**Solución:**
1. Verifica tu conexión a internet
2. Limpia la cache del navegador
3. Intenta acceder desde otro navegador
4. Reporta el problema si persiste
