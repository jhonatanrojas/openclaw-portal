# 👤 Guía de Usuario - OpenClaw Portal

## 🏠 Dashboard Principal

### 📊 Panel de Control
El dashboard principal muestra:
1. **Estadísticas clave**: Documentación, agentes, sistema, actividad
2. **Acciones rápidas**: Accesos directos a funciones comunes
3. **Documentación reciente**: Últimos artículos creados o modificados
4. **Actividad del sistema**: Eventos recientes y logs

### ⚡ Acciones Rápidas Disponibles
- **Nueva Documentación**: Crear nuevo artículo de documentación
- **Ver Agentes**: Acceder a la gestión de agentes
- **Estado del Sistema**: Verificar salud del portal
- **Configuración**: Ajustes de perfil y sistema

## 📚 Gestión de Documentación

### Crear Nueva Documentación
1. Navegar a **Documentación** en el menú principal
2. Hacer clic en **+ Nueva Documentación**
3. Completar los campos:
   - **Título**: Nombre descriptivo
   - **Categoría**: Instalación, Configuración, API, Agentes, etc.
   - **Contenido**: Usar formato Markdown
   - **Versión**: Versión del documento (ej: 1.0)
   - **Estado**: Activo/Inactivo

### Editor Markdown
El editor soporta:
- **Formato básico**: Negrita, cursiva, encabezados
- **Listas**: Ordenadas y no ordenadas
- **Código**: Bloques de código con resaltado de sintaxis
- **Enlaces e imágenes**: Inserción de recursos
- **Vista previa**: Previsualización en tiempo real

### Organización y Búsqueda
- **Filtros por categoría**: Click en categorías para filtrar
- **Búsqueda**: Barra de búsqueda en la parte superior
- **Paginación**: Navegación entre páginas de resultados
- **Ordenamiento**: Por fecha de actualización

## 🤖 Gestión de Agentes

### Ver Agentes Conectados
1. Navegar a **Agentes** en el menú principal
2. Ver lista de agentes activos e inactivos
3. Información mostrada:
   - **Nombre**: Identificador del agente
   - **Estado**: Activo/Inactivo/Error
   - **Última conexión**: Fecha y hora
   - **Sesión**: ID de sesión activa

### Enviar Mensajes a Agentes
1. Seleccionar un agente de la lista
2. Hacer clic en **Enviar Mensaje**
3. Escribir el mensaje en el editor
4. Hacer clic en **Enviar**

### Monitoreo de Actividad
- **Logs en tiempo real**: Ver actividad reciente
- **Métricas**: Uso de recursos por agente
- **Alertas**: Notificaciones de errores

## ⚙️ Configuración del Sistema

### Perfil de Usuario
1. Hacer clic en el nombre de usuario (esquina superior derecha)
2. Seleccionar **Perfil**
3. Campos editables:
   - **Nombre**: Nombre completo
   - **Email**: Dirección de correo
   - **Contraseña**: Cambiar contraseña
   - **Avatar**: Imagen de perfil

### Configuración de OpenClaw
1. Navegar a **Configuración** → **OpenClaw**
2. Configuraciones disponibles:
   - **Workspace Path**: Ruta del workspace de OpenClaw
   - **Autenticación**: Método de autenticación
   - **Backups**: Configuración de backups automáticos
   - **Notificaciones**: Alertas y notificaciones

### Preferencias de la Aplicación
- **Tema**: Claro/Oscuro
- **Idioma**: Español/Inglés
- **Zona horaria**: Configuración regional
- **Formato de fecha**: Preferencias de visualización

## 🔧 Herramientas Avanzadas

### Editor de Archivos
1. Navegar a **Herramientas** → **Editor de Archivos**
2. Explorar estructura del workspace
3. Funcionalidades:
   - **Crear archivos/directorios**
   - **Editar contenido**
   - **Subir/descargar archivos**
   - **Buscar y reemplazar**

### Terminal Web
1. Navegar a **Herramientas** → **Terminal**
2. Ejecutar comandos directamente
3. Características:
   - **Sesiones persistentes**
   - **Historial de comandos**
   - **Autocompletado**
   - **Multiples pestañas**

### Monitor de Recursos
1. Navegar a **Monitor** → **Recursos**
2. Métricas en tiempo real:
   - **Uso de CPU**: Porcentaje de utilización
   - **Memoria**: Uso y disponibilidad
   - **Disco**: Espacio utilizado/libre
   - **Red**: Tráfico de entrada/salida
